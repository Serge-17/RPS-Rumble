# RockPaperScissors-Rumble

RockPaperScissors-Rumble — это простое и увлекательное приложение, реализующее классическую игру "Камень, Ножницы, Бумага". Игра позволяет пользователям сразиться с компьютером, выбирая один из трех вариантов и пытаясь обыграть противника.

<img src="RPS Rumble/Assets.xcassets/RPS Rumble.png" alt="Simulator Screenshot - iPhone 16 Pro" width="300" />

## Возможности

- **Простой интерфейс**: Удобный и интуитивно понятный интерфейс для выбора варианта игры.

- **Мгновенные результаты**: Пользователи могут сразу увидеть результат игры после каждого раунда.

- **Поддержка нескольких раундов**: Игра позволяет проводить несколько раундов, что добавляет элемент соревнования.

- **Статистика**: Приложение отображает статистику побед, поражений и ничьих.
